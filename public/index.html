PR Quality Intelligence

ChunkCheck helps developers understand whether a GitHub pull request is ready for review.

The website analyzes any GitHub Pull Request and returns a quality score from 0 to 100. It checks files changed, lines changed, test presence, CI status, documentation signals, and commit quality, then gives clear suggestions like adding tests or splitting a large PR.
